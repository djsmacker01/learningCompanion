API Reference

Authentication

All API requests require authentication using your API key.

Authorization: Bearer YOUR_API_KEY
Base URL
https://api.learningcompanion.com/v1
Endpoints
Method Endpoint Description
GET /topics Get user's topics
POST /topics Create a new topic
GET /sessions Get user's study sessions
POST /sessions Start a new study session
PUT /sessions/{id} Update a study session
GET /analytics Get user analytics
Example Request
curl -X GET "https://api.learningcompanion.com/v1/topics" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
Example Response
{
  "topics": [
    {
      "id": "123",
      "title": "Machine Learning",
      "description": "Introduction to ML concepts",

    }
  ],

  "page": 1,
  "per_page": 20
}

Developer Guide

SDK Installation
# Python
pip install learning-companion-sdk

# JavaScript
npm install learning-companion-sdk

# PHP
composer require learning-companion/sdk
Python Example
from learning_companion import LearningCompanion

# Initialize client
client = LearningCompanion(api_key='YOUR_API_KEY')

# Create a topic
topic = client.topics.create(
    title='Python Programming',
    description='Learning Python basics'
)

# Start a study session
session = client.sessions.start(
    topic_id=topic.id,
    duration_minutes=25
)

# Complete the session
client.sessions.complete(
    session_id=session.id,
    confidence_after=8
)
JavaScript Example
import LearningCompanion from 'learning-companion-sdk';

// Initialize client
const client = new LearningCompanion({
  apiKey: 'YOUR_API_KEY'
});

// Create a topic
const topic = await client.topics.create({
  title: 'JavaScript Fundamentals',
  description: 'Learning JS basics'
});

// Start a study session
const session = await client.sessions.start({
  topicId: topic.id,
  durationMinutes: 25
});

// Complete the session
await client.sessions.complete({
  sessionId: session.id,
  confidenceAfter: 8
});
